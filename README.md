## 技术栈

#### 后端 Golang 1.19

- Gin 1.9.1             [(Web框架)](https://gin-gonic.com/zh-cn/)
- GORM v1.9.16     [(ORM)](https://gorm.io/zh_CN/)
- MySQL 5.7             [(数据库)](https://www.mysql.com/)


#### 已实现的功能
- 抽奖后端的增删改查和抽奖执行
- 企业微信机器人随机发送图片
